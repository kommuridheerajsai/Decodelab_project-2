#  Expense Tracker

A simple Python-based Expense Tracker that allows users to enter multiple expenses, calculate the total amount spent, count the number of expenses, and find the average expense.

This project is built as part of **Python Programming Project 2 - DecodeLabs Industrial Training Kit**.

---

##  Project Overview

The Expense Tracker is a beginner-friendly Python project designed to understand how numerical data is collected, processed, and calculated in real time.

The user can continuously enter expense amounts. The program keeps adding each expense to the total using accumulator logic and displays a final report when the user finishes.

---

##  Objective

The main objective of this project is to create a Python script where users can:

- Enter expense amounts one by one
- Add all expenses automatically
- View the total amount spent
- Count the number of expenses entered
- Calculate the average expense

---

##  Features

- Add multiple expense amounts
- Calculate total expenses
- Count total number of expenses
- Calculate average expense
- Stop the program using `done` or `quit`
- Handles invalid input
- Prevents negative expense values
- Displays a clean final report

##  How It Works

1. The program starts with total expense as `0`.
2. The user enters an expense amount.
3. The input is converted into a number.
4. The expense amount is added to the total.
5. The expense count is increased.
6. The program continues until the user types `done` or `quit`.
7. Finally, the program displays:
   - Total expenses
   - Number of expenses
   - Average expense
